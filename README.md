## 部署数据库实例

1. 在`localhost`启动相应的数据库实例，并创建相应的数据表
2. 在`configuration`配置类中设置数据库访问连接串，使程序能够连接到数据库实例

## 启动应用程序

1. 通过`SpringBoot`启动应用程序
2. 通过`http`访问应用程序接口进行调试


1. 通过`test`单元测试进行调试
